7.3 Password Strength Check

Algorithm:

1. Input password.

2. Check if length >= 8.

3. Check if contains at least one uppercase letter.

4. Check if contains at least one lowercase letter.

5. Check if contains at least one digit.

6. Check if contains at least one special character (!@#$%^&*()).

7. If all conditions are true, output "strong", else output "weak".

output:

weak

strong
